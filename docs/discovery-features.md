# AI Logic for Vibe Matching

The AI matching system focuses on compatibility, energy, and shared context, not appearance.

### 1. Vibe Vector Scoring
Each user has a dynamic Vibe Vector calculated from:
*   Selected interests
*   Prompt answers
*   Room participation
*   Nudges sent/received
*   Time-of-day activity patterns

### 2. Context-Aware Matching
Matching logic adapts based on intent and time:
*   **Daytime:** Academic & productivity vibes.
*   **Evenings:** Social & hangout vibes.
*   **Weekends:** Fun & exploration vibes.

### 3. Compatibility Over Popularity
The algorithm avoids showing only â€œpopularâ€ profiles:
*   Prioritizes mutual relevance.
*   Boosts quieter but compatible users.
*   Avoids repetitive profile exposure.

### 4. AI Prompt Alignment
Users are matched based on how similarly they answer prompts.
*   *Example:* Two users choose â€œStaying inâ€. AI checks why (reading, gaming, rest) and matches users with aligned intent.

### 5. Micro-Feedback Learning
The AI continuously learns from:
*   Ignored vs. accepted nudges.
*   Time spent in rooms.
*   Repeat interactions.

### 6. Exploration vs. Comfort Balance
The system balances:
*   **Comfort matches:** Very similar vibes.
*   **Exploration matches:** Slightly different but complementary vibes to avoid echo chambers.
